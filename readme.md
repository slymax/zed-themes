**Zedokai** is a theme for [Zed](https://zed.dev) based on the [Monokai Pro](https://monokai.pro) color scheme

> [!TIP]
> You can enable improved syntax highlighting by running
> `curl -s https://slymax.com/zedokai/syntax.sh | sh`
